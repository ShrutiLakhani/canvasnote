
# CanvaNote

 
 ### A Note taking app to plan your daily tasks and ideas better
 
 ### Features
1. User Signup
2. User Login
3. Add a Note with
  - Priority
  - Labels like Home, WOrk, Personal etc
  - Choose a color for the note
4. Edit a Note
5. Archive a Note 
6. Trash a Note
7. Filter & Sort notes added on
  - Priority
  - Date/Time added
  - Labels

### Built with
 - HTML & CSS
 - React JS
